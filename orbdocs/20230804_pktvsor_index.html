get /api/v1/metrics/app
Retrieve global application information (getGlobalAppInfo)

Return type

AppInfo

Example data

Content-Type: application/json
{
  "app" : {
    "single_summary" : true,
    "periods" : 6.027456183070403,
    "up_time_min" : 1.4658129805029452,
    "deep_sample_rate_pct" : 0.8008281904610115,
    "version" : "3.0.7"
  },
  "dns" : {
    "xact" : {
      "open" : 10
    }
  }
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Global application information AppInfo

500

server error

get /api/v1/metrics/window/{periodCount}
Retrieve a merged window of metrics over several periods (getMergedWindow)

Path parameters

periodCount (required)
Path Parameter — The number of merged periods to retrieve, from 2 to the number of available periods

Return type

inline_response_200_1

Example data

Content-Type: application/json
{
  "{periodCount}m" : {
    "dns" : {
      "xact" : {
        "in" : { },
        "out" : { }
      },
      "wire_packets" : { },
      "cardinality" : { }
    },
    "packets" : {
      "rates" : {
        "pps_in" : {
          "p99" : 2.3021358869347655,
          "p90" : 5.962133916683182,
          "p50" : 1.4658129805029452,
          "p95" : 5.637376656633329
        }
      },
      "top_ipv4" : [ {
        "name" : "name",
        "estimate" : 7.061401241503109
      }, {
        "name" : "name",
        "estimate" : 7.061401241503109
      } ],
      "cardinality" : {
        "dst_ips_out" : 6.027456183070403,
        "src_ips_in" : 0.8008281904610115
      }
    }
  }
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Metrics output inline_response_200_1

500

server error

get /api/v1/metrics/bucket/{period}
Retrieve metrics for an individual (single) 60s period (getPeriodBucket)

Path parameters

period (required)
Path Parameter — The specific single 60s period to retreat metrics for, numbered from 0 (which is the most recent and actively being updated) to the number of available periods - 1

Return type

inline_response_200

Example data

Content-Type: application/json
{
  "1m" : {
    "dns" : {
      "xact" : {
        "in" : { },
        "out" : { }
      },
      "wire_packets" : { },
      "cardinality" : { }
    },
    "packets" : {
      "rates" : {
        "pps_in" : {
          "p99" : 2.3021358869347655,
          "p90" : 5.962133916683182,
          "p50" : 1.4658129805029452,
          "p95" : 5.637376656633329
        }
      },
      "top_ipv4" : [ {
        "name" : "name",
        "estimate" : 7.061401241503109
      }, {
        "name" : "name",
        "estimate" : 7.061401241503109
      } ],
      "cardinality" : {
        "dst_ips_out" : 6.027456183070403,
        "src_ips_in" : 0.8008281904610115
      }
    }
  }
}
